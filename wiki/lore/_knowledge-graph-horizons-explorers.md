# Knowledge Graph: Horizons, Explorers, Ancient Adventure, and Amber Use

Author-facing relationship map.

Goal:
- make the Horizons-derived material easy to reason about while writing *Pokemon Amber*
- separate clean canon anchors from Amber-specific adaptation choices
- provide a reusable node/edge map rather than loose notes

See also:
- `wiki/lore/_pokemon-horizons-reference.md`
- `wiki/lore/explorers.md`

### Practical read

Horizons has three important structural lanes:
1. **ancient expedition myth**
2. **modern adventure crew travel**
3. **modern antagonistic organization exploiting ancient discovery**

That triptych is one of the most useful things to borrow for Amber.

## 2. Amber Adaptation Graph

### Broad role / profession layer

- **explorers** (lowercase) -> profession / trainer lane
- **Rangers** -> official route-safety / wilderness-response institution
- **League trainers** -> badge / gym / conference institution

### Broad role edges

- **explorers -> specialize in -> off-route rescue / retrieval / mapping / investigation / discovery**
- **explorers -> overlap with -> quest board economy**
- **explorers -> associated with -> mystery dungeons / abnormal spaces**
- **explorers -> distinct from -> Rangers**
- **explorers -> can become -> famous route-openers / region-link discoverers**

### Capitalization / naming edge

- **Explorers** (capitalized) -> Horizons-specific named group
- **explorers** (lowercase) -> Amber continuity profession / social class

This overlap is intentionally tolerated.

## 3. Ancient Adventure Pattern Worth Preserving

### Pattern

- ancient figure or small expedition party
- hidden destination / rumored paradise / lost settlement / unknown route
- exceptional companion Pokemon or scattered old allies
- strange material or relic tied to the destination
- modern descendants / inheritors / readers of the old trail
- modern institutions trying to exploit or control the old discovery

### Why it matters for Amber

This pattern is already close to things Amber wants to do:
- hidden places
- old routes
- buried history
- relic substances / strange stones
- modern organizations trying to capitalize on old discoveries
- Pokemon companions that outlive their age and become historical anchors

Amber does not need to copy Horizons directly, but this pattern is extremely reusable.

## 4. Material-Lore Pattern Worth Preserving

### Horizons pattern

- **Laquium** is not just treasure
- it is:
  - mythic
  - scientific
  - dangerous
  - exploitable
  - politically and personally important

### Amber analog use

When Amber uses strange ore, relic stones, or ancient materials, they work best when they have at least 3 of these dimensions:
- practical use
- mythic significance
- corruption / danger / instability risk
- institutional interest
- emotional history

This is one reason the Charcadet stone / future Mewtwo thread works.

---

## 5. Travel / Expedition Pattern Worth Preserving

### Horizons pattern

- ship-based or expedition-based mobility
- a stable group with differentiated jobs
- navigation is a real role
- movement across regions is normalized
- anime-original places are allowed to matter

### Amber use

Even if Amber stays more road-based than Horizons, writing should remember:
- travel can involve logistics, not just montage
- teams can have differentiated field roles
- some people travel to win badges, some to discover, some to research, some to salvage
- the world is not closed at the edge of the game map

## 6. Continuity-Handling Graph

### Safe continuity edges

- **Ash -> still exists somewhere in the world**
- **Horizons -> does not clearly pin exact timeline distance from Ash**
- **Horizons -> allows time to move more than classic Ash anime often did**
- **Amber continuity -> may age characters up if useful**

### Writing consequence

Treat Horizons as:
- shared-world compatible
- exact-date fuzzy
- broad-lore useful
- adaptable rather than binding in every detail

## 7. Region-Expansion Use Graph

This is especially important for Amber.

### Node chain

- **explorers**
- **mystery dungeons / abnormal spaces**
- **hidden settlements / rumored routes**
- **new regional connections**
- **old expedition records**
- **modern rediscovery**

### Key edge logic

- explorers -> clear / survive -> abnormal spaces
- abnormal spaces -> block -> normal travel / normal commerce
- successful explorer parties -> map -> usable paths
- mapped paths -> reveal -> hidden settlements / region links / old ruins
- rediscovered paths -> justify -> adding more places to the story later

This is one of the strongest reasons to formalize explorers in Amber continuity.

---

## 8. Things to Keep Separate

To avoid muddying the setting, keep these distinctions clear:

### A. Explorer profession vs named Horizons faction
- lowercase profession != capitalized organization

### B. Ordinary dangerous places vs true mystery dungeons
- not every cave is a mystery dungeon
- not every rescue is explorer work

### C. Ancient adventure legend vs modern job-board labor
- Lucius-style ancient-adventurer myth is the legendary high end
- everyday explorers still need money, clients, logistics, and working identities

### D. Wonder vs exploitation
- old discoveries create wonder
- modern institutions try to industrialize them
- that tension is part of what makes the setting feel modern rather than fairy-tale simple

## 10. Short Version

If the repo ever needs a one-paragraph reminder:

**Horizons proves Pokemon is now comfortable with ancient adventurer legends, world-mystery travel crews, hidden destinations, dangerous special substances, anime-original geography, and named exploration organizations. Amber can safely adapt that by making explorers a real trainer lane, while preserving the canon-specific Explorers as a separate named historical group.**
